# NEXUS
NEXUS será o Software-Mestre que utilizará Módulos para acompanhar evolução dos pacientes.
Terá Módulos como:
A) iMeddis -> responsável por Identificar, classificar, pontuar e dar o remanejo clínico recomendado ao paciente para evitar as interações graves e prevenir reações adversas.
B) LabPro -> responsável por tudo de Análises Clínicas do paciente. Terá os dados de resultados de exames de todos os tipos, como de Hematologia, Bioquímica, Imunologia, Urinálise, Parasitologia etc. Ele terá Integração total com o iMeddis, por se tratar de dados necessários para o cálculo das interações medicamentosas e cruzar com os dados biométricos do paciente.
C) EndoInject -> responsável pela Terapia de Injetáveis do paciente, tendo total integração com os outros módulos (LabPro, iMeddis, OdontoFlow, 'Photo'go)
D) OdontoFlow -> responsável pela área de Odontologia do paciente, desde a gestão e marcação de consultas, mas também de todas as outras recomendações devidas ao paciente sobre tudo que ele precisa saber.
E) 'Photo'go -> Área de imagem, IMAGENOLOGIA: exames de imagem como Ultrassom (de todas as áreas), Raio-X (de todas as áreas), Ressonância Magnética, Doppler e todas as suas variações, mais a Tomografia.
